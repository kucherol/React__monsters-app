# React Monsters App

This is a small pet project on React course.

Please check: https://react-monsters-app.vercel.app/

This project goes over all of the fundamentals of React including...

- Components
- JSX
- Props (proptypes, defaultprops, etc)
- State (Component & App Level)
- Styling
- Handling Events
- Lists & Keys
- Forms
- Context API
- HTTP Requests

# Usage

### Install dependencies

```bash
npm install
```

### Run

```bash
npm run dev
```
